# Jsp + Servlet + JavaBean MVC小例子

用户提交用户名和密码给后台，后台去数据库中匹配，

如果匹配到了，那么返回该用户的用户名，密码，所读学校，身高等信息。

这里用到了MySql数据库，动态生成了页面。

利用了MVC三层模型
